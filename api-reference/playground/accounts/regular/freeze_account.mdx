---
title: "Freeze/Unfreeze Account"
description: "This endpoint is used to send a request to freeze a customer's account."
api: "POST /account/bank/freeze"
---

<Info>
	- When the freeze field is passed as '**true**' the account would be frozen, updating the status to inactive. 
	- When the freeze field is passed as '**false**' the account status would be reverted to active.
</Info>

<ParamField header="requestId" type="string" required />
<ParamField body="accountNumber" type="string" required>
	The account number that a freeze is to be done against
</ParamField>
<ParamField body="reason" type="string" required>
	The reason for which the freeze is requested
</ParamField>
<ParamField body="freeze" type="boolean" default={true} required />

<ResponseExample>

```json
{
	"statusCode": 00,
	"message": "Account successdully frozen",
	"data": {},
	"errors": []
}
```

</ResponseExample>
