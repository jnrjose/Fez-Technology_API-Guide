---
title: "Close/Re-Open Account"
description: "This endpoint is used to send a request to close a customer's account."
api: "POST /account/close"
---

<Info>
	- When the close field is passed as '**true**' the account would be closed,
	updating the status to closed. - When the close field is passed as '**false**'
	the account status would be reverted to active.
</Info>

<ParamField header="requestId" type="string" required />
<ParamField body="accountNumber" type="string" required>
	The customer's account number
</ParamField>
<ParamField body="reason" type="string" required>
	The reason for request
</ParamField>
<ParamField body="close" type="boolean" default={true} required />

<ResponseExample>

```json
{
	"statusCode": 00,
	"message": "Account successfully closed",
	"data": {},
	"errors": []
}
```

</ResponseExample>
